Description

Part 1 | Calculating Course Grade

A program that takes a student's overall homework grade, mid-term exam score, final exam score, and participation grade to produce their overall grade in the course. The grade is based on Percentage Based Grade Scale (Ex: 90-100 - A). If the grade is below a C, the student is notified to retake the course. Only valid input values are processed by the program. An invalid value will result in an error message.

Part 2 | jQuery UI

A program that calculates a given sales employee's monthly earnings based on the number of certain items they've sold for the month. Base pay is $200 + 9% of gross sales for the week. In addition, when hovering over the Salesperson's Name Text Field, the message "Please make sure that the salesperson's name is spelled correctly" is displayed. This message is implemented using the jQuery UI tooltip.

Part 3 | Temperature Conversion

A program that converts a Temperature, given in Fahrenheit, to Celsius, or vice versa. The program accepts all real numbers (-∞ to ∞), and presents the apprioprate tempatue as a whole intger. The user is responsible the choosing the correct conversion button to get the apprioprate temparture. Inputs outside of real numbers, will result in an error message.

Part 4 | Elementary Multiplication Quiz

A program that will help an elementary school student learn multiplication. The student then types the answer into an input box. Your program checks the student's answer. If it is correct, display the message "Very good!" in a text area or another part of the page, and then prompt the user to see whether they wish to continue by using the confirm statement. If true, then generate a new question. If not, then exit the script and do not present another question.
